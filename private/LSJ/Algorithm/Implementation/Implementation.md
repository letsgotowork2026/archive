# 구현
=> 피지컬로 승부하는 문제

## 정수형 표현 범위
- Java: BigInteger 표준 라이브러리
- C++ : 외부 라이브러리 검색 필요 => longlong 이상의 문제는 잘 출제되지 않음
- 파이썬 : 자료형 지정할 필요 없음, 매우 큰 수 연산 기본 지원

## 리스트 크기
- 보통 코테 메모리 제한 : 128 ~ 512MB

|데이터 개수|메모리 사용량|
|---:|:---:|
|1,000|약 4KB|
|1,000,000|약 4MB|
|10,000,000|약 40MB|

- 리스트 크기, 개수 유의하기
- 코테에서는 입출력 문제도 있기 때문에 대체로는 메모리 제한 안에서 가능

## PyPy vs Python3
- PyPy3 : 파이썬3 문법 지원, 실행속도 빠름
- 대략 c, c++ 수준의 실행속도
